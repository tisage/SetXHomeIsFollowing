## English Description
Tired of being hijacked by social media algorithms? This lightweight userscript ensures that X (formerly Twitter) always defaults to the **Following** tab instead of the "For You" algorithm feed.

### Features
- **Auto-Switch**: Automatically clicks the "Following" tab when you visit X/Twitter home.
- **SPA Support**: Works seamlessly even when navigating back to home from other pages (Notifications, Profile, etc.) without a page refresh.
- **Multilingual**: Supports both English ("Following") and Chinese ("æ­£åœ¨å…³æ³¨").
- **Fast & Light**: Zero dependencies, optimized for performance.
